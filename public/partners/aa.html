SOS × Aid Arena

Partnership Scope — March 2026

Aid Arena has built a trusted, vetted network of disaster response nonprofits — AidBase. SOS has built AI-powered coordination infrastructure that makes every action measurable. Together, we solve engagement and scale — at the same time.

What SOS Brings

  • Measurable coordination — every match, deployment, and referral produces data funders can see
  • AI agents in Slack — no new platform to learn, the agent meets you where you work
  • Privacy by design — each partner's data stays separate, PII encrypted and masked
  • Intelligence that compounds — every disaster makes the system smarter

What Aid Arena Brings

  • AidBase — a vetted network of active organizations from Team Rubicon to ToolBank
  • Coordination hub — where responders already go to connect
  • Real-world validation — active partners, real workflows, board-level backing
  • Training backbone — Collab Ops Academy gets real data to center around

SOS Today (MVP)

  • AI agents for citizen intake and partner coordination — live in Slack or on the web
  • 100-point matching engine — scores proximity, capability, urgency, trust
  • Operational masking — PII encrypted, only shared after bilateral consent
  • Decision traces — every AI decision auditable, every outcome tracked
  • 3 partner agents deployed — Emergency RV, Free Hot Meals, Aid Arena
  • Hurricane Helene data — 25 citizens, 50+ matches, full lifecycle tested

Where SOS Is Going

  • Per-partner databases — each org owns their data, SOS owns the intelligence
  • N-way coordination chains — multi-org sequenced responses (transport → shelter → food)
  • Network intelligence — coordinator orgs see their entire network's performance
  • Learning flywheel — every disaster makes the system smarter, weight changes proposed + approved
  • Multi-channel — WhatsApp, iMessage, SMS, Slack, web — config, not code
  • CaaS licensing — coordination kernel applied to healthcare, workforce, food banks

How It Works

1
Partner signs up through Aid Arena — one-click authentication
2
Gets their own SOS agent in Slack or on the web dashboard
3
Agent onboards them — confirms capabilities, coverage area, capacity
4
When a disaster hits — citizens submit needs through any channel
5
SOS matches automatically — scores proximity, capability, urgency, trust
6
Partner accepts in one message — "Accept" in Slack or the dashboard
7
Coordination happens in a shared thread — all participants visible
8
Impact report generated — Aid Arena sees their entire network's performance
